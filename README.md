# AeroCMS Astro Starter

Astro starter website for Aero CMS using `@aero-cms/astro-sdk`.

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

## Pages

- `/` — Home
- `/haberler`, `/etkinlikler`, `/dokumanlar`
- `/iletisim`, `/arama`, `/[slug]`
